
Logic

1. Take inputs for coefficients a, b, and c.

2. Compute the discriminant (D) using:
   D = b*b - 4*a*c

3. Check conditions:

- If D > 0 → Two real and distinct roots.

- If D == 0 → One real and equal root.

- If D < 0 → Two complex (imaginary) roots.

4. The program uses a custom sqrt_approx() function (Newton’s method) to approximate square roots.
