Logic

1. Start the program.
2. Declare a character array str [100] for input string.
3. Read the string using fgets() to include spaces.
4. Display a menu with options:
   - Count vowels
   - Count consonants
   - Count digits
   - Exit
5. Use switch-case inside a do-while loop for the menu.
6. For each case, traverse the string character-by-character and count as required.
7. Display the result.
8. Repeat until user chooses to Exit.
9. End the program.
