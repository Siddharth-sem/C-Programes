Logic

1. Start the program.
2. Display a menu with three options:
   - Matrix Addition
   - Matrix Subtraction
   - Exit
3. Take the user's choice.
4. If the choice is 1 or 2,
   - Take the number of rows and columns.
   - Input elements for both matrices.
5. Use if-else inside the switch to perform either:
   - Addition: result [i] [j] = matrix1[i][j] + matrix2[i][j]
   - Subtraction: result[i][j] = matrix1[i][j] matrix2[i][j]
6. Display the resultant matrix.
7. Repeat until user selects Exit (3).
8. End the program.
