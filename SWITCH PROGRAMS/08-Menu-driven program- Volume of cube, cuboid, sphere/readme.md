Logic

1. The program displays a menu with 3 options:

- 1 → Cube

- 2 → Cuboid

- 3 → Sphere

2. The user selects a shape.

3. Depending on the choice, it asks for dimensions (like side, radius, etc.).

4. The program calculates and displays the volume using the following formulas:

- Cube: side³

- Cuboid: length × width × height

- Sphere: (4/3) × π × radius³

5. If an invalid option is chosen, an error message is displayed.
