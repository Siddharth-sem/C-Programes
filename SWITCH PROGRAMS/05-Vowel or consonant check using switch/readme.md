Logic

1. The program takes a single character input from the user.

2. Using a switch-case statement, it checks if the character is one of the vowels (a, e, i, o, u or uppercase equivalents).

3. If not a vowel, it checks whether the input is a valid alphabetic character.

4. Displays:

- "Vowel" if the character is a vowel.

- "Consonant" if it’s an alphabet but not a vowel.

- "Invalid input" if it’s not a letter.
