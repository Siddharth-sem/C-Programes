Logic

1. Start the program.

2. Declare variables:
choice, principal, rate, time, interest, and total_amount.

3. Display the menu:
- Option 1 → Simple Interest
- Option 2 → Compound Interest

4. Take user input for choice.
- If invalid, display an error and terminate.

5. Ask the user for:

- Principal amount
- Rate of interest
- Time period
Each input must be positive — validate accordingly.

6. Use a switch-case statement:

- Case 1:
Calculate Simple Interest using:
SI=(P×R×T)/100
Display SI and total amount.

- Case 2:
Calculate Compound Interest using:
CI=P×((1+R/100)T-1)

7. Display CI and total amount.

8. Stop the program.
