Logic

1. Start the program.

2. Declare variables: choice, num1, num2, result, and cont.

3. Use a do-while loop to repeatedly display the menu.

4. Display the menu options (1–5).

5. Take user input for choice.

6. If choice is valid (1–5):

- Ask the user to enter two integers.

- Use switch(choice) to perform the corresponding operation:

   - Case 1 → Addition

   - Case 2 → Subtraction

   - Case 3 → Multiplication

   - Case 4 → Division (check for zero)

   - Case 5 → Modulus (check for zero)

7. If choice is invalid, show an error message.

8. Ask the user if they want to continue (y/n).

9. Repeat if the user enters y or Y.

10. End the program.
