1. The program displays a menu of shapes:

-  → Circle
-  → Rectangle
-  → Triangle

2. The user enters a choice number.

3. Based on the choice, the program:

- Takes necessary dimensions (radius, length, etc.)
- Checks if they are positive.
- Calculates the area using:
- Circle → π × r²
- Rectangle → length × breadth
- Triangle → ½ × base × height

4. Displays the result with two decimal precision.

5. Handles invalid inputs gracefully
