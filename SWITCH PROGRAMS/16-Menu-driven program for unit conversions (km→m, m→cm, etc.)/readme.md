Logic

1. Start the program.

2. Declare variables: choice, value, and result.

3. Display the conversion menu:

- 1 → Kilometers to Meters

- 2 → Meters to Centimeters

- 3 → Centimeters to Millimeters

- 4 → Meters to Kilometers

- 5 → Exit

4. Take user input for choice.

5. If choice is between 1–4, ask for the input value to convert.

6. Use a switch statement:

- Case 1: Convert km → m using result = value * 1000

- Case 2: Convert m → cm using result = value * 100

- Case 3: Convert cm → mm using result = value * 10

- Case 4: Convert m → km using result = value / 1000

- Case 5: Exit the program

- Default: Display invalid choice message

7. Use an if condition to ensure entered value ≥ 0.

8. Display the converted value.

9. Stop the program.
