Logic

1. The program takes marks (0–100) as input.

2. It checks if the input is valid:

- If marks < 0 or > 100 → prints “Invalid marks”.

3. If valid, it divides marks by 10 and uses switch(marks / 10) to determine the grade:

- 90–100 → A

- 80–89 → B

- 70–79 → C

- 60–69 → D

- 50–59 → E

- 40–49 → E--

- Below 40 → F

4. Displays the corresponding grade on the screen.
